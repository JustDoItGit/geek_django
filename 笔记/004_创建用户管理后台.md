1、安装包
```shell
pip install django
```
2、创建一个项目  
recruitment 项目名称
```shell
django-admin startproject recruitment
```
3、启动项目
```shell
cd recruitment
python ./manage.py runserver 0.0.0.0:8000
```
访问网站：127.0.0.1:8000

4、创建后台用户库
```shell
python manage.py makemigrations
```
没有创建
